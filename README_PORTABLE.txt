GESTION JAUGES — VERSION PORTABLE

1. Copiez tout le contenu de ce patch dans C:\GestionJauges.
2. Remplacez les fichiers lorsqu'on vous le demande.
3. Double-cliquez sur CREER_PORTABLE.bat.
4. Le résultat sera créé dans : C:\GestionJauges\Portable\GestionJauges.exe

Cette version ne s'installe pas. Elle utilise la même base Supabase que la version React et le Setup.

Important : Windows peut afficher un avertissement SmartScreen parce que l'exécutable n'est pas signé numériquement. Dans ce cas, utilisez « Informations complémentaires » puis « Exécuter quand même » uniquement si la politique informatique de votre entreprise l'autorise.
